# NanoGPTX

A decoder-only Transformer (GPT-style) language model implemented completely from scratch using PyTorch.

## Objectives

- Build every core GPT component from first principles.
- Understand how modern LLMs work internally.
- Train a small language model on open datasets.
- Produce a professional, open-source engineering project.

## Features

- Character-level tokenizer
- Dataset loader
- Token embeddings
- Positional encoding
- Multi-head self-attention
- Feed-forward network
- Layer normalization
- Transformer decoder blocks
- Autoregressive text generation
